This is a minimal Next.js starter configured to read Supabase env vars:

- `NEXT_PUBLIC_SUPABASE_URL`
- `NEXT_PUBLIC_SUPABASE_ANON_KEY`
- `SUPABASE_SERVICE_ROLE`
- `REKONET_MODE`
- `EMAIL_DRY_RUN`

## Local dev
```
npm i
npm run dev
```

## Deploy
Push this repo to GitHub and import into Vercel. Ensure env vars are set in Vercel project settings.
